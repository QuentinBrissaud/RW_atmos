# RW_atmos
Solver of dispersion equations for RW-induced acoustic waves

# Requirements
- Pyhton3.7
- Python modules: 
  - numpy==1.18.5
  - pandas==1.1.2
  - matplotlib==3.2.1
  - pyrocko==2020.8.18
  - scipy==1.5.2
  - obspy==1.2.2
  - sympy==1.6.2
- gcc

# To install
1. retrieve the code
2. move to the root directory
3. run "mkdir Linux.m64"
4. run make earthsr
5. chmod u+x earthsr

# To run
- Use Python notebook "nb_RW_atmos.ipynb"
- Use the Python routine "one_RW_atmos.py"
